# Bank-Customer-Churn-Prediction-ANN

🏦 Bank Customer Churn Prediction using Artificial Neural Network (ANN)

This project predicts whether a bank customer is likely to leave the bank using an Artificial Neural Network (ANN) built with TensorFlow and Keras.

# Features

- Trained on a bank customer churn dataset
- Data preprocessing using Pandas and StandardScaler
- ANN model with multiple hidden layers
- Predicts whether a customer will leave the bank or stay
- Calculates the total number of customers likely to churn and the total number likely to stay
- Supports uploading a new Excel file for prediction in Google Colab
- Displays prediction results and overall customer counts

# Technologies Used

- Python
- Pandas
- Scikit-learn
- TensorFlow / Keras
- Google Colab

# Workflow

1. Load the dataset.
2. Preprocess the data using StandardScaler.
3. Split the dataset into training and testing sets.
4. Train the ANN model.
5. Evaluate the model on test data.
6. Upload a new Excel file for prediction.
7. Predict customer churn and display:
   - Customers who will leave the bank
   - Customers who will stay with the bank
   - Total number of customers
