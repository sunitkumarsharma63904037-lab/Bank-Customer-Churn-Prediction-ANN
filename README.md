## 🌟 <span style="color:#00F5A0;">Project Overview</span>

This project uses an **Artificial Neural Network (ANN)** to predict whether a customer will leave the bank (**churn**) or stay.

---

## 🧠 <span style="color:#38BDF8;">Model Architecture</span>

```
Input (8 Features)
   ↓
Dense(32) → ReLU
   ↓
Dense(16) → ReLU
   ↓
Dense(8) → ReLU
   ↓
Dense(4) → ReLU
   ↓
Output(1) → Sigmoid
```

---

## ⚙️ <span style="color:#FACC15;">Key Features</span>

- 📊 Data Scaling (StandardScaler)  
- 🤖 ANN Model (Keras)  
- 🎯 Binary Classification  
- 📈 Accuracy Evaluation  
- 📂 Excel Upload Prediction
